Javascript Student Grade Generator, Speed Detector, and Net Salary Calculator

  1. The first program written in the grade.js file generates a string of students grade based on user input of student marks, expressed and passed in as an argument to the function.
  2. The second program written in the speed.js file takes in the speed of a car passed in as an argument to the function. It then generates respective strings based on the relationship of the user speed to the speed limit, 70km/s.
  3. The third program written in the salary.js file generates an indovidual's Net Salary based on basic salary and benefits passed into the function as arguments.
